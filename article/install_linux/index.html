Install Linux

by IICT Team

Posted on Fri, Feb 19, 2016

Motivation

Most of the labs you will be working on during your stay are meant to be realised on a Unix-based system. That’s why it’s a good idea to prepare a Linux VM, that you will use whenever you need it.

If you work on Windows or Mac OS X, you will need to use a Linux VM. We recommend you to work on Ubuntu.

Note: the following steps are for those who need a linux VM. You can use VirtualBox, VMware, Xen, etc but we strongly recommend that you use VirtualBox, as there are a few settings we need to change and this document will only talk about the VirtualBox configuration. Download VirtualBox.

Create an Ubuntu Desktop VM

Usually, you can get two types of ISOs when you want to install an Ubuntu image:

  • Server version
  • Desktop version

The main difference is, the Ubuntu server version is meant to be used as a server (who would have guessed) so there is no GUI. The programs and packages are also different, since both versions don’t have the same aim (would be unwise to use a desktop version as a server). That’s why we recommend that you install the desktop version.

  1. Download Ubuntu Desktop, version 14.04 LTS 64 bit from the Ubuntu website or the swiss mirror. You need to pick the desktop version, which should be called ubuntu-14.04-desktop-amd64.iso.

  2. Install your VM with your virtualization software. Pick the iso you previously downloaded as the source (also called CD image). If you are using VMware, don’t use the “easy install” option.

    • Choose language: leave default (English)

    • Installation options: Install Ubuntu (Install Ubuntu)

    • Erase Disk and install Ubuntu (Install Now)

    • Where are you (Zurich) (Should be by default)

    • Configure the keyboard: Switzerland > Switzerland - Switzerland - French (Switzerland)

    • Set up users and passwords - A user account will be created for you:

      • Full name: your full name
      • Your computer’s name: default
      • Username for your account: administrator
      • Password: invent a password
      • Encrypt your home directory?: leave default (No)

    • Finish the installation: Continue

    The installation finishes and the VM reboots. After booting you should see the Linux login prompt.
