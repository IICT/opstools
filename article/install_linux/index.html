Install Linux on a VM

by IICT Team

Posted on Fri, Feb 19, 2016

Motivation

Developers and system administrators often need to re-create on their personal workstation the server environment they are working on. Servers often run Linux as operating system while for personal workstations Windows or Mac OS X is prevalent. Having a separate workstation for the server OS or installing the server OS via multi-boot is a hassle. Creating a virtual machine provides a quick way to run a new OS alongside the existing one. This document will show you how to install Ubuntu Linux in a virtual machine.

Create an Ubuntu Desktop VM

Usually, you can get two types of ISOs when you want to install an Ubuntu image:

  • Server version
  • Desktop version

The main difference is that the Ubuntu server version is meant to be used on server machines, so there is no graphical user interface whereas the desktop version has a GUI and ist meant to be used on a personal workstation. The server version has fewer packages than the desktop version and is approximately half the size.

Normally experienced developers and sytem administrators install the server version because they will work on the command line and don’t need the overhead of the GUI. As you are only getting started with the command line it is helpful to have the GUI in the beginning, so we recommend you install the desktop version.

  1. Download Ubuntu Desktop, version 16.04.2 LTS 64 bit from the Ubuntu website or the swiss mirror. You need to pick the desktop version, which should be called ubuntu-16.04.2-desktop-amd64.iso.

  2. Install your VM with your virtualization software. Depending on your virtualization software configure the ISO file you previously downloaded (also called CD image) as the disk in the virtual optical drive, or pick it as the source of the new VM. If you are using VMware, don’t use the “easy install” option.

    • On the first screen leave the language at the default (English). Then choose Install Ubuntu.

    • Preparing to install Ubuntu: Check Download updates

    • Installation Type: Select Erase Disk and install Ubuntu.

    • Where are you?: Select Zurich (Should be selected by default)

    • Configure the keyboard: Select Switzerland > Switzerland - Switzerland - French (Switzerland)

    • Who are you? - This will create a user account for you:

      • Your name: your full name
      • Your computer’s name: leave at default
      • Pick a username: leave at default
      • Password: invent a password
      • Encrypt your home directory?: leave default (No)

    • Finish the installation: Continue

    The installation finishes and the VM reboots. After booting you should see the Ubuntu login screen.
