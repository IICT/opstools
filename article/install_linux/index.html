Install Linux on a VM

by IICT Team

Posted on Fri, Feb 19, 2016

Motivation

Developers and system administrators often need to re-create on their personal workstation the server environment they are working on. Servers often run Linux as operating system while for personal workstations Windows or Mac OS X is prevalent. Having a separate workstation for the server OS or installing the server OS via multi-boot is a hassle. Creating a virtual machine provides a quick way to run a new OS alongside the existing one. This document will show you how to install Ubuntu Linux in a virtual machine.

Create an Ubuntu Desktop VM

Usually, you can get two types of ISOs when you want to install an Ubuntu image:

  • Server version
  • Desktop version

The main difference is, the Ubuntu server version is meant to be used on server machines, so there is no graphical user interface whereas the desktop version has a GUI and ist meant to be used on a personal workstation. The server version has fewer packages than the desktop version and is approximately half the size.

Normally experienced developers and sytem administrators install the server version because they will work on the command line and don’t need the overhead of the GUI. As you are only getting started with the command line it is helpful to have the GUI in the beginning, so we recommend you install the desktop version.

  1. Download Ubuntu Desktop, version 14.04 LTS 64 bit from the Ubuntu website or the swiss mirror. You need to pick the desktop version, which should be called ubuntu-14.04-desktop-amd64.iso.

  2. Install your VM with your virtualization software. Pick the iso you previously downloaded as the source (also called CD image). If you are using VMware, don’t use the “easy install” option.

    • Choose language: leave default (English)

    • Installation options: Install Ubuntu (Install Ubuntu)

    • Erase Disk and install Ubuntu (Install Now)

    • Where are you (Zurich) (Should be by default)

    • Configure the keyboard: Switzerland > Switzerland - Switzerland - French (Switzerland)

    • Set up users and passwords - A user account will be created for you:

      • Full name: your full name
      • Your computer’s name: default
      • Username for your account: administrator
      • Password: invent a password
      • Encrypt your home directory?: leave default (No)

    • Finish the installation: Continue

    The installation finishes and the VM reboots. After booting you should see the Linux login prompt.
