Control your AWS expenses

by IICT team

Posted on Wed, Feb 24, 2016

AWS gives to students a generous credit, but if you are not careful and leave unused resources running, the credit can disappear quickly. You should check your remaining credit at least once a week using the procedure described in the following. Log into your AWS account. Using the drop-down menu with your name (upper right) navigate to Billing & Cost Management. Under Spend Summary you should see a big $0.00. If you see another amount than $0.00 something is very wrong, because you spent all your credit.

Quick guide to Markdown

by IICT team

Posted on Wed, Feb 24, 2016

What is Markdown Markdown is a very lightweight markup language (from which you can produce HTML) that is being used by a lot of people. It’s fast, efficient, really easy to read and compatible pretty much everywhere. You write a Markdown file in a text editor and it looks almost like a normal formatted text file. There are many websites and examples where the text is written in Markdown: READMEs and documentation files on github are in Markdown (extension .md) Some of the documentation on Moodle is written in Markdown.
