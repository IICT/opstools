The VirtualBox tools are meant to be installed on a VM, to enable some options that are not possible otherwise. Allow SSH on your VM (Optionnal) Install Git bash for Windows users only You will need to use the ssh command from your Windows host on your guest VM. There are some tools like Putty that allows you to do so, but we will install Git bash instead. Git bash is an emulation of the BASH terminal and allows you (more or less) to use Linux commands from a terminal on a Windows host.
