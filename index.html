Control your AWS expenses

by IICT team

Posted on Wed, Feb 24, 2016

AWS gives to students a generous credit, but if you are not careful and leave unused resources running, the credit can disappear quickly. You should check your remaining credit at least once a week using the procedure described in the following. Log into your AWS account. Using the drop-down menu with your name (upper right) navigate to Billing & Cost Management. Under Spend Summary you should see a big $0.00. If you see another amount than $0.00 something is very wrong, because you spent all your credit.

Share folders with your VM using only the command line

by IICT team

Posted on Wed, Feb 24, 2016

Installing VirtualBox Guest Additions (on a server version) You need to install the VirtualBox Guest Additions to use different useful tools, like shared folders, mouse integration or the full-screen support. On a server edition though, the only thing we want is the ability to work with shared folders. Installing the tools Start by Going in the VirtualBox menu on your VM, go to Devices -> Insert Guest Additions CD image… Install the following dependencies : $ sudo apt-get update && sudo apt-get install dkms build-essential linux-headers-$(uname -r) Mount the CD: $ sudo mount /dev/cdrom /media/cdrom # You may have a read-only warning, it's normal, CD are by definition read-only devices.
